## Installing Step Forge

### Required Dependencies

Install from npm:

```bash
npm install -D @step-forge/step-forge
```

### Optional Dependencies

You'l probably want to use an assertion library, we recommend [Earl](https://www.npmjs.com/package/earl)

```bash
npm install -D earl
```
